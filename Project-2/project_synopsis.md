# Project Synopsis

## Title
Credit Risk Default Prediction

## What Are We Building?
We are building a machine learning system that predicts whether a loan applicant will default in the next 12 months.

## Why This Project Is Useful
Banks and finance companies face losses when customers fail to repay loans. This model helps identify risky applicants before loan approval.

## Data Used
The system uses:
- Applicant details
- Credit bureau details
- Previous loan history
- Payment behaviour
- Credit card usage

## Final Output
The model generates:
- Default probability
- Predicted default class

## Business Impact
This project supports credit underwriting, risk management, and better loan approval decisions.
