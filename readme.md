Project Description:
This project builds a personalized course recommendation system using machine learning. 
It recommends online courses based on user skills, interests, course category, difficulty level, platform, and ratings.

Dataset Used:
Coursera.csv / Online Courses Across Platforms Dataset

Technologies Used:
Python
Pandas
NumPy
Scikit-learn
TF-IDF Vectorizer
Cosine Similarity
MinMaxScaler
Pickle

Recommendation Techniques:
1. Content-Based Filtering
2. Skill-Based Recommendation
3. Hybrid Recommendation System

Evaluation Metrics:
Precision@10: 0.8
Recall@10: 0.8
F1-Score@10: 0.8
MAP@10: 0.8
NDCG@10: 0.866

How to Run:
1. Open the project folder in VS Code.
2. Make sure Coursera.csv is in the same folder as task1.py.
3. Run the command:
   python task1.py
